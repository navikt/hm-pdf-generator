# hm-pdf-generator
Applikasjon for konvertering av html dokumenter til PDF 

# Komme i gang

Gradle brukes som byggverktøy og er bundlet inn.

```
./gradlew build
```


## Kjøre lokalt
Start applikasjonen i Idea
